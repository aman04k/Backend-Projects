React, also known as React.js or ReactJS, is a popular open-source front-end JavaScript library used for building user interfaces, particularly for single-page applications. It was created by Jordan Walke, a software engineer at Meta (formerly Facebook). The initial prototype of React was called "F-Bolt" before it was later renamed to "FaxJS"

.

React was first deployed on Facebook's News Feed in 2011 and later integrated into Instagram in 2012. The project was officially open-sourced in May 2013 at JSConf US, marking a significant milestone in its adoption and growth

.

React is maintained by Meta and a community of individual developers and companies. It has evolved significantly since its inception, with major updates and new features being introduced regularly. For instance, React 16.8 introduced React Hooks, which allow developers to use state and other React features without writing a class
